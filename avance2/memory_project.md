Avance 2 del proyecto UNSHOT Bar completado. Incluye diagrama E-R (5 tablas), script SQL para Supabase (PostgreSQL), y aplicación Java EE con JSP + Servlets para insertar y listar datos de clientes y bebidas.

**Why:** Requisito académico del avance 2: ER + BD + inserción + listado en JSP/Java.

**How to apply:** El proyecto está en avance2/. La BD está en Supabase (no MySQL local). El driver a usar es postgresql-42.7.3.jar. Conexion.java necesita el HOST y PASSWORD de Supabase del estudiante.
